# casino  
  
## Server Requirements  
* python3
* python3-dev
* virtualenv
* pip3
* apache2
* libapache2-mod-wsgi-py3
* libmysqlclient-dev
  
## Update app on server  
To update the app on the server from this repo, execute the following on the server:  
```
sh /var/www/html/clone_permissions_venv.sh
```
